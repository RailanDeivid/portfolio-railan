PRINCIPAIS PROJETOS.

Análise Roubos de veículos no municipio
de Guarulhos-SP

Este projeto faz parte de uma atividade extensiva dedicada à análise e disseminação de dados públicos sobre furtos e roubos em Guarulhos-SP. A iniciativa visa não apenas compreender as tendências de crimes na região, mas também proporcionar insights valiosos para a comunidade. Através da aplicação de técnicas avançadas de ciência de dados.

Scraping anuncios de veículos
do MercadoLivre e previsão de preço de vendas.

Este projeto realiza web scraping de anúncios de veículos no Mercado Livre e a previsão de valores de venda utilizando um modelo de machine learning. As etapas incluem a coleta de dados por meio de técnicas de web scraping, seguida pela transformação e limpeza dos dados para garantir consistência e adequação ao modelo. Em seguida, é realizada uma análise exploratória para identificar padrões e tendências que possam influenciar os preços dos veículos. Em seguida foi desenvolvido e treinado um modelo de machine learning para prever os valores de venda com base nas características extraídas dos anúncios, e por fim o deploy do modelo usando Streamlit.

Automatização de Extração de Dados
da CEAGESP.

Neste projeto, apresento uma solução desenvolvida para automatizar a extração de dados do site de cotações de preços de entrepostos da CEAGESP. Essa tarefa manual, que envolvia a obtenção da média de preços de venda no atacado, era realizada às segundas, quartas e sextas-feiras, exigindo intervenção manual constante.

O principal objetivo deste processo é capturar as cotações de preços dos entrepostos, permitindo análises futuras sobre os preços no atacado.

Análise Exploratória de Dados da API
do YouTube.

O projeto visa realizar uma análise exploratória de dados utilizando informações de vídeos do YouTube. este projeto aborda conceitos e técnicas de ciência de dados, aplicando-os especificamente à análise de vídeos na plataforma YouTube.

O objetivo principal é explorar a API do YouTube para obter dados detalhados sobre vídeos e analisar o que influencia o desempenho de um vídeo na plataforma.

Análise exploratória e Dataviz do conjunto de dados Olist.

Neste projeto, realizo uma análise exploratória de um conjunto de dados sobre comércio eletrônico brasileiro, especificamente sobre pedidos realizados na Olist Store. Utilizando um conjunto de dados público que abrange 100 mil pedidos feitos entre 2016 e 2018, este projeto tem como foco explorar a fundo a visualização de dados.
